#+title: rugit – Project Tasks

* Tasks

** TODO Line-level staging
Stage/unstage individual lines within a hunk (visual selection or cursor-based). The UX model: expand a hunk, move cursor to a line, press =s= to stage just that line. Requires splitting hunks in the backend and applying partial patches.

** TODO Branch management popup (=b=)
Magit-style transient: =b b= checkout, =b c= create+checkout, =b d= delete, =b r= rename. Needs a branch list picker UI.

** TODO Interactive rebase UI (=r i=)
Show rebase todo list (pick/reword/edit/squash/fixup/drop per commit). Arrow keys to reorder, keybinds to change action. Drives =git rebase -i= via a custom sequence editor.

** TODO Instant fixup/squash (=c F= / =c s=)
Magit-style: pick a commit from a list, create a fixup or squash commit targeting it, optionally auto-rebase immediately.

** TODO Diff viewer buffer (=d= on a commit or file)
Full-screen diff view, navigable by hunk. Invoked from log buffer (=d= on selected commit) or status buffer (=d= on a file). Syntax-highlighted diff lines.

** TODO Log buffer improvements
Current log buffer is minimal. Add: commit graph rendering (=--graph=), author/date columns, filtering (=l f=), searching (=/=), pressing =Enter= to open diff viewer.

** TODO Stash popup (=z=)
=z z= stash, =z p= pop, =z a= apply, =z d= drop, =z l= list stashes. Needs stash list UI.

** TODO Rebase/merge menus (=r=, =m=)
=r r= rebase onto branch, =r e= rebase --continue, =r a= abort. =m m= merge, =m a= abort.

** TODO Extended vim bindings in commit editor
=o=/=O= open line below/above, =w=/=b=/=e= word motions, =G=/=gg= jump to end/start, =u=/=C-r= undo/redo, =yy=/=p=/=P= yank and paste line.

** TODO Colour themes (catppuccin, gruvbox, nord)
Theme enum in config.toml, palette lookup table, pass palette through render calls.

** TODO jj backend – full implementation
Implement all Backend trait methods using =jj= subprocess calls.

** TODO Packaging (Homebrew formula, AUR PKGBUILD, Nix flake)
